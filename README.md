# 📖 Project Overview

Financial markets are influenced not only by numerical indicators but also by news, corporate announcements, economic events, and investor sentiment. While Machine Learning models perform well using structured numerical data, they often cannot capture the qualitative context that influences market behaviour.

This project investigates how **Natural Language Processing (NLP)** can be applied to financial text to extract meaningful analytical signals through **sentiment analysis, entity recognition, and thematic analysis**.

It represents the **second stage** of a four-project analytical journey, extending structured financial analysis into unstructured textual information.

---

# 🎯 Business Problem

Traditional financial analysis primarily relies on structured numerical datasets. However, financial markets also react to qualitative information that cannot be represented through numerical variables alone.

This project explores whether Natural Language Processing can transform financial text into structured analytical signals that improve business understanding.

---

# ❓ Research Question

> **Can Natural Language Processing extract meaningful analytical insights from financial text that complement structured financial market data?**

---

# 🎯 Project Objectives

- Analyse financial news using Natural Language Processing.
- Measure sentiment within financial text.
- Identify important entities influencing financial markets.
- Discover recurring financial themes and market narratives.
- Evaluate whether textual information provides additional analytical understanding beyond structured market data.

# 📈 Results

The project demonstrated that different Natural Language Processing techniques contribute differently to financial analysis.

Sentiment analysis alone produced limited predictive value, while thematic analysis captured broader market narratives more effectively.

Entity recognition provided useful contextual information but was less influential than recurring financial themes.

---

# 🔍 Key Findings

- Sentiment alone is not sufficient for understanding market behaviour.
- Financial themes produced stronger analytical signals than isolated news events.
- Entity recognition improved contextual understanding but contributed limited standalone predictive value.
- Combining multiple NLP techniques provides richer analytical insight than relying on a single text metric.
- Unstructured financial text complements structured financial data rather than replacing it.

---

# 💼 Business Impact

This project demonstrates how Natural Language Processing can transform qualitative financial information into structured analytical insights.

Rather than viewing financial news as unstructured text, organizations can use NLP techniques to identify recurring themes, market sentiment, and contextual signals that support better analytical decision-making.
